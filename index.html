HISTORY

I've been curious about the magic box from a very young age. I spent a lot of time playing games and editing photos to make them look better. This led me to try Blender (though I never got very good at it) and eventually to programming. I fell in love with solving problems through code when I was around 17.


During the covid 19 pandemic, I finished high school but was stuck in a box with no friends around. That's when I discovered Linux, a new friend. After some hesitation, I gave it a try and it showed my old potato laptop a new life. This is where my real journey started.


I began with C programming and learned the basics. I explored front-end development and UI design, then tried Rust but found it too difficult. So I went back to C to strengthen my roots. I built some small projects, learned about memory management and working with strings, and created a structure for future projects. I made a simple puzzle game and an SVG image generator using what I grasped about parsing and string manipulation. Now I am excited about the obstracles along the way. I'm planning to rewrite everything in rust though :^)
